aₙ の式（nは1始まり）

このページが「分かる」形
まず「周期mごとに、nについて次数d以下の多項式（係数は剰余で変わってOK）」を検出できれば、 Hurwitzゼータの ζ(-p,q) を使って F(0) を計算します。
それがダメなら「aₙ ≈ C·n^α」をログ回帰で検出し、F(0)=C·ζ(-α) を数値で評価します（べき関数フォールバック）。
すぐ試せる式例
  • 1 → -1/2
  • (n mod 2 == 1) ? 1 : 0（1,0,1,0,…）→ 0
  • (n mod 2 == 1) ? n : 0 → 1/12
  • n^5 または n**5 → ζ(-5) = -1/252
  • sqrt(n) → ζ(-1/2)（べきモデル側で出ます）
  • ビット遊び: bxor(i32(n), 5)
注意: このページの ^ は累乗扱いです（XORは bxor を使います）。 mod や == などは mathjs の文法です。
