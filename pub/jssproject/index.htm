* JSProject *

このページは？

LLMたちに習ったりAntigravityを活用しながら実験しているJavaScript作品置き場です。
再利用などはご自由にどうぞ。教えてくれるとちょっとウレシイです。

ゲーム系

ゲームっぽい体験のできるJavaScript作品です。

ASCII TET

3つのモードを持つASCII画面のテトリスです。

Topowalk

ヘンな位相空間上を歩くゲームです。特に無目的。clockwiseはちょっとヤバいです。

Hungle Typing Game

ハングルの単語や短文をタイピングするゲームです。

Tarot trick-taking game

パパユーというゲームを遊べます。

Integrated mahjong system

変則的な麻雀ゲームを遊べます。UIをもうちょっと吟味したい。今は観戦が主です。

音系

音声生成、加工、可視化などを扱うJavaScript作品です。

Formula Synth

数式から波形を生成するシンセサイザーです。音律変更が利きます。
キーボードで演奏機能はPCブラウザ限定です。

Formula Synth+ADSRF

Formula SynthのADSR+フィルター版。クラドニ図形が見られます。

TE Tuning Calc

TE Tuning, POTE Tuningなどの調律法を計算します。分からない人はXenharmonic.wikiを閲覧のこと。 Prime Basis Qに求めたい素数、Mapping Vにステップ数を入れてください。

Spectro Sampler

wavまたは画像を読み込んでスペクトログラムを弄れます。まだ未完成です。

Glitch MIDI

midiファイルを読み込んでバグらせる。イルーム音楽とかお好きなら……。音楽は　脳が腐りそうよ。

EDO Scale Calculator

MoS Scaleの解析。シンボルはLsだけじゃなくて、L≧M≧N≧...≧sとたくさん使えます。MoS Scale同士の足し算も出来ます。 クリップボードにコピーしてMathSynthとかに貼ってもいいかも。

JI Explorer

純正音階探索機。LMNO...sの語形式にも出来るので、右上のEDO Scale Calculatorとの合わせ技（得られた純正音階をMOSで近似するとか）もあるかと思います。 精密なconvexキャプチャが効かないのと、純正音階をクリップボードにコピーが出来ないためそこはこれから。

画像系

画像を加工したり、視覚効果を試したりできるJavaScript作品です。

8bit Matrix

方眼上に並んだ数字が、文字色と背景色を激しく変更します。スマホOK。

Polyline & Blur

折れ線をランダム描画→ぼかし、を繰り返すデモ。PCブラウザ推奨。

Imagebound

画像を読み込ませると揺らぐデモ。マザー2の戦闘画面が元ネタ。PCブラウザ推奨。

MathPattern Img Drawer

数式を与えると画像を生成。スマホOK。

Hex Tiling

画像を読み込ませると万華鏡のような画像を生成します。スマホOK。

Penrose Tiling

画像を読み込ませるとペンローズタイルに描き込みます。スマホOK。

Glitch Tool

与えられた画像をバグっぽく加工します。

 上書き用素材があるので、こちらを読み込ませてよしなに。

Glitch Matrix

画像を読み込ませると、与えられた画像を16×16の方眼で分割し、色調を変えながら再配置。

geometricVJ

数式から与えられた画像のパレットや模様を連続変化。

Palette Animator

画像を読み込ませてパレットを回転させてジラジラさせる。簡易なペイント機能もあります。

Rectangle Tile Generator

画像を組み合わせてタイルテクスチャを作成。目地処理あり。縦横比を変えられるので床フローリングなどにも。また、行か列の末尾に整数を書くと（a b 5みたいに）それぞれズラせます。

GIF Merger

アニメーションGIFをレイヤー的に統合します。上は透過させてね。

Geometric VJ 2

重たくなって新登場！ ……ではなく、GIF出力+Earthboundっぽいディストーションが出来るようになりました。

Glitch Clock

有意味と無意味のあわいを漂う7セグデジタル時計。

テキスト系

テキストをいじったりして効果を得るJavaScript作品です。

Glitch Poet

入力された文字列をバグらせます。真価は詳細モード＋「異世界参照」オン＋「異世界データ」に無関係なものを入れること（必殺技名や捻った一言などがおすすめ）。己自身の意表すらも突く表現を作ってみませんか？

Glitch Converter

あらゆるものをバイナリ化して色んな表現をします。

Multi-Keyboard Notepad

入力支援。画面上のキーボードを見ながら韓国語、ロシア語、ギリシャ語のタイピングが出来ます。

Binary Glitch Laboratory

Hex Editorでbyte列と構造を見ながら、画像・音声・テキストの実データ領域を確率的に操作する学習用ラボです。

数学系

数学勉強のためのJavaScript作品です。

Eliptic Vurve Visualizer

楕円曲線の和やn倍点を図示します。有限体モードもあります。
