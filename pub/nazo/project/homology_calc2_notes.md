# High Dimensional Homology Calculator notes

`homology_calc2.html` は、有限単体複体のホモロジーをブラウザ上で計算するための観察用ツールです。

主な目的は、単体複体を入力して

- 境界写像 `∂n`
- 境界行列
- 係数体 `GF(2)` / `Q` 上の Betti 数
- 整数係数 `Z` 上の自由部分とねじれ成分

を見比べることです。

## 入力形式

基本形は次の通りです。

```text
Simplices:
[0,1,2]
[0,1,3]
[0,2,3]
[1,2,3]
```

`[0,1,2]` は 2 単体、つまり三角形を表します。
`[0,1,2,3]` は 3 単体、つまり四面体を表します。

最大単体だけを書けば、そのすべての面は自動で追加されます。
たとえば `[0,1,2,3]` を入力すると、頂点、辺、三角形面も内部的にすべて追加されます。

## 実装の全体像

計算は次の流れで進みます。

1. 入力文字列から単体を取り出す。
2. 各単体のすべての面を自動生成する。
3. 次元ごとに単体を分類する。
4. 境界行列 `∂n : C_n -> C_{n-1}` を作る。
5. 係数に応じて階数や Smith 標準形を計算する。
6. `H_n = Ker ∂n / Im ∂(n+1)` を表示する。

## 主な関数

### `parseComplex(text)`

入力テキストから単体複体を作る関数です。

`[0,1,2]` のような記法を正規表現で拾い、各単体の全ての面を `choose` で生成します。
戻り値は、おおまかには次のような形です。

```js
{
  byDim: [
    [["0"], ["1"], ["2"]],
    [["0","1"], ["0","2"], ["1","2"]],
    [["0","1","2"]]
  ],
  maxDim: 2
}
```

`byDim[n]` が `n` 単体の一覧です。

### `boundaryMatrix(cx, n)`

境界写像 `∂n : C_n -> C_{n-1}` の行列を作ります。

単体 `[v0,v1,...,vn]` の境界は

```text
∂[v0,...,vn] = Σ (-1)^i [v0,...,vhat_i,...,vn]
```

です。ここで `vhat_i` は `vi` を除くという意味です。

例:

```text
∂[0,1,2] = [1,2] - [0,2] + [0,1]
```

`GF(2)` 係数では `-1 = +1` になるので、符号の違いは消えます。

### `rankOverField(M, field)`

`GF(2)` または `Q` 上で行列の階数を計算します。

- `GF(2)` では成分を 0/1 にして掃き出し法を行います。
- `Q` では `Frac` クラスで有理数を正確に扱います。

浮動小数点数を使わないので、小さい例では丸め誤差が起きません。

### `Frac`

有理数を表す小さなクラスです。

```js
new Frac(2, 4) // 1/2 に約分される
```

`add`, `sub`, `mul`, `div` を持ち、`Q` 上の掃き出し法で使われます。

### `snfWithColumnTransform(M, fallbackCols)`

整数行列の Smith 標準形を計算します。

Smith 標準形とは、整数の行基本変形と列基本変形によって

```text
diag(d1, d2, ..., dr)
```

という対角形にする操作です。

ただし `homology_calc2.html` では、単に対角成分を得るだけではなく、列変換行列も追跡しています。
これは整数係数のホモロジーでねじれ成分を正しく出すために必要です。

戻り値には主に次が含まれます。

- `diag`: Smith 標準形の非零対角成分
- `rank`: 非零対角成分の数
- `V`: 列変換行列
- `Vinv`: 列変換行列の逆行列
- `normal`: 変形後の行列

### `integerHomologyGroups(cx, mats)`

整数係数 `Z` のホモロジー群を計算する中心部分です。

やりたいことは

```text
H_n = Ker ∂n / Im ∂(n+1)
```

です。

係数体上なら

```text
dim H_n = dim Ker ∂n - dim Im ∂(n+1)
```

だけで Betti 数が出ます。

しかし `Z` 上では、商群に `Z/2Z` のような有限巡回群が現れることがあります。
これがねじれ成分です。

この関数では次の手順を取ります。

1. `∂n` の Smith 標準形を取り、列変換から `Ker ∂n` の整数基底を得る。
2. `Im ∂(n+1)` の各列を、その `Ker ∂n` 基底上の座標に変換する。
3. その座標行列に Smith 標準形をかける。
4. 対角成分 `2, 3, ...` から `Z/2Z`, `Z/3Z`, ... を表示する。

これにより、たとえば射影平面の

```text
H_1(RP^2; Z) = Z/2Z
```

が表示できます。

### `analyze(cx, field)`

選択された係数に応じて全次元のホモロジーを計算します。

- `GF2` / `Q`: 境界行列の階数から Betti 数を計算
- `Z`: `integerHomologyGroups` で自由部分とねじれ成分を計算

### `groupText(g, field)`

計算結果を表示用の文字列に整形します。

例:

```text
Z
Z^2
Z/2Z
Z ⊕ Z/3Z
F2^2
Q
0
```

### UI 関連の関数

`runAll()`
: 入力を読み取り、ホモロジー群の表を表示します。

`showBoundary()`
: 境界写像を式として表示します。

`showMatrices()`
: 境界行列を表示します。

`checkComplex()`
: 単体の面がすべて入っているかを確認します。現在は最大単体から自動補完するため、通常は成功します。

`draw()`
: 複体を 2 次元に投影して簡易表示します。高次元の形そのものを描くものではなく、頂点と辺、低次元面の雰囲気を見るためのものです。

## 計算例

以下では Betti 数を

```text
(β0, β1, β2, ...)
```

で書きます。

### 点

```text
Simplices:
[0]
```

結果:

```text
H_0 = Z
Betti = (1)
```

1つの連結成分だけがあります。

### 円 `S^1`

```text
Simplices:
[0,1]
[1,2]
[2,0]
```

結果:

```text
H_0 = Z
H_1 = Z
Betti = (1, 1)
```

連結成分が1つ、1次元の穴が1つです。

### 塗りつぶした三角形 `D^2`

```text
Simplices:
[0,1,2]
```

結果:

```text
H_0 = Z
H_1 = 0
H_2 = 0
Betti = (1, 0, 0)
```

円周の穴は、三角形の面によって埋められます。

### 球面 `S^2`

四面体の表面です。

```text
Simplices:
[0,1,2]
[0,1,3]
[0,2,3]
[1,2,3]
```

結果:

```text
H_0 = Z
H_1 = 0
H_2 = Z
Betti = (1, 0, 1)
```

1次元の穴はありませんが、2次元の空洞が1つあります。

### 四面体そのもの `D^3`

```text
Simplices:
[0,1,2,3]
```

結果:

```text
H_0 = Z
H_1 = 0
H_2 = 0
H_3 = 0
Betti = (1, 0, 0, 0)
```

中身まで詰まった立体なので、1点に縮められる、つまり可縮です。

### 3球面 `S^3`

4単体の境界です。

```text
Simplices:
[0,1,2,3]
[0,1,2,4]
[0,1,3,4]
[0,2,3,4]
[1,2,3,4]
```

結果:

```text
H_0 = Z
H_1 = 0
H_2 = 0
H_3 = Z
Betti = (1, 0, 0, 1)
```

これは高次元版の球面らしい結果です。
最下位の `H_0` と最上位の `H_3` だけが残ります。

### 4球面 `S^4`

5単体の境界です。

```text
Simplices:
[1,2,3,4,5]
[0,2,3,4,5]
[0,1,3,4,5]
[0,1,2,4,5]
[0,1,2,3,5]
[0,1,2,3,4]
```

結果:

```text
H_0 = Z
H_4 = Z
Betti = (1, 0, 0, 0, 1)
```

この例は `homology_calc2.html` が 3 次元より上も扱えることを確認するのに向いています。

### くさび和 `S^1 ∨ S^2`

円と球面を1点でくっつけた空間です。

```text
Simplices:
[0,1]
[1,2]
[2,0]
[0,3,4]
[0,3,5]
[0,4,5]
[3,4,5]
```

結果:

```text
H_0 = Z
H_1 = Z
H_2 = Z
Betti = (1, 1, 1)
```

円の1次元穴と、球面の2次元空洞が同時に残ります。

### トーラス `T^2`

結果:

```text
H_0 = Z
H_1 = Z^2
H_2 = Z
Betti = (1, 2, 1)
```

トーラスには独立な1次元ループが2本あります。
1本はドーナツの穴をぐるっと回る方向、もう1本はチューブの周りを回る方向です。

またトーラスは向き付け可能な閉曲面なので、`H_2 = Z` が現れます。

### 射影平面 `RP^2`

プリセット `射影平面 RP^2` で確認できます。

整数係数:

```text
H_0 = Z
H_1 = Z/2Z
H_2 = 0
```

有理数係数:

```text
H_0 = Q
H_1 = 0
H_2 = 0
Betti = (1, 0, 0)
```

`GF(2)` 係数:

```text
H_0 = F2
H_1 = F2
H_2 = F2
Betti = (1, 1, 1)
```

ここがとても面白いところです。

整数係数では `H_1 = Z/2Z` という 2-ねじれが見えます。
有理数係数では、この有限部分は消えます。
一方 `GF(2)` 係数では、2で割った世界なのでこの構造が見え方を変えて現れます。

## 数学的な理論

### チェイン群

単体複体 `K` に対して、`n` 次元単体を形式的な基底として作る自由アーベル群を `C_n(K)` と書きます。

たとえば辺が3本あるなら

```text
C_1 ≅ Z^3
```

です。

係数を `Q` や `GF(2)` に変えた場合は、自由ベクトル空間として考えます。

### 境界写像

境界写像

```text
∂n : C_n -> C_{n-1}
```

は、各 `n` 単体をその境界に送ります。

重要な性質は

```text
∂n ∘ ∂(n+1) = 0
```

です。

「境界の境界は空」という直感に対応します。

### サイクルとバウンダリ

`Ker ∂n` の元を `n`-サイクルと呼びます。
境界を持たない `n` 次元の輪です。

`Im ∂(n+1)` の元を `n`-バウンダリと呼びます。
これは1つ上の次元のものの境界として現れる輪です。

ホモロジーは

```text
H_n = Ker ∂n / Im ∂(n+1)
```

です。

つまり、「境界を持たない輪」のうち、「何かの境界として埋められるもの」を 0 と見なしたものです。

### Betti 数

体上で計算すると、ホモロジー群はベクトル空間になります。
その次元を Betti 数と呼びます。

```text
βn = dim H_n
```

直感的には次のように読めます。

- `β0`: 連結成分の数
- `β1`: 独立な1次元ループの数
- `β2`: 独立な2次元空洞の数
- `β3`: 独立な3次元空洞の数

ただしこれはあくまで直感です。
高次元では「穴」という言葉だけでは足りなくなりますが、サイクルとバウンダリの商として考えると統一的に扱えます。

### ねじれ成分

整数係数では、ホモロジー群はベクトル空間ではなくアーベル群です。
そのため

```text
Z
Z^2
Z/2Z
Z/3Z
Z ⊕ Z/2Z
```

のような形が現れます。

`Z^r` の部分が自由部分で、`r` が Betti 数です。
`Z/kZ` の部分がねじれ成分です。

たとえば

```text
H_1(RP^2; Z) = Z/2Z
```

は、2回回ると境界になって消えるが、1回だけでは消えないループがある、という意味に対応します。

### Smith 標準形

整数行列 `A` に対して、整数の可逆行列 `P`, `Q` を用いて

```text
P A Q = diag(d1, d2, ..., dr)
```

という形にできます。
これを Smith 標準形と呼びます。

この対角成分から、商群の構造が読めます。

```text
Z^m / image(A)
```

のような群は、Smith 標準形により

```text
Z^(m-r) ⊕ Z/d1Z ⊕ ... ⊕ Z/drZ
```

のように分解できます。

`homology_calc2.html` では、`Ker ∂n` の中で `Im ∂(n+1)` がどのように入っているかを整数座標で表し、その行列に Smith 標準形を適用しています。

## 係数による違い

同じ空間でも、係数を変えると見え方が変わります。

### `Q`

有理数係数では、有限ねじれ成分は見えません。

```text
Z/2Z ⊗ Q = 0
```

なので、`RP^2` の `H_1` は `Q` 係数では消えます。

### `GF(2)`

`GF(2)` では `-1 = +1` です。
向きの符号が消えるため、非向き付け可能な空間の最上位ホモロジーが見えることがあります。

`RP^2` で

```text
H_2(RP^2; GF(2)) = GF(2)
```

となるのはその代表例です。

### `Z`

整数係数はもっとも情報が多い設定です。
自由部分に加えて、ねじれ成分も見えます。

ただし計算は体上より難しくなります。
階数だけでは足りず、Smith 標準形が必要になります。

## 注意点

このツールは教育用・観察用です。
大きな複体では行列サイズが急に大きくなり、Smith 標準形の計算が重くなることがあります。

また、可視化はあくまで2次元投影です。
高次元複体の幾何そのものを正確に表示しているわけではありません。

## 眺めると楽しい実験

1. `S^1` と `D^2` を比べる。
   円の `H_1` が、面を貼ると消えることが見えます。

2. `S^2` と `D^3` を比べる。
   球面の `H_2` が、中身を入れると消えます。

3. `S^3` や `S^4` を見る。
   高次元球面でも、最上位ホモロジーだけが残ることが見えます。

4. `T^2` を見る。
   `H_1 = Z^2` という、2本の独立ループが見えます。

5. `RP^2` で係数を切り替える。
   `Z`, `Q`, `GF(2)` の違いが最もよく見えます。
