差分から分数階差分へ

連番の和の拡張と、正則化をめぐる自習ノート

このノートの目標は、整数個の連番の和 \[ \sum_{m=a}^{a+k-1} m = ka+\frac{k(k-1)}{2} \] を出発点として、差分作用素と分数階差分の立場から 「\(k\) を有理数や実数に拡張すると何が起きるか」を整理することである。

大事な方針

分数階差分をそのまま二項級数で書くと、増大する関数では発散が起きる。 したがって、

  • 発散の様子を見る道具としては Abel 正則化や指数カットオフを使う
  • 実際の値を与える定義としては、下降階乗冪 \(x^{\underline{\beta}}\) を基底にした定義を採る

という二段構えにするのが見通しがよい。

· · ·

§0. 出発点：連番の和

\(a\) から始まる \(k\) 個の連番の総和は

\[ S(k,a)=\sum_{m=a}^{a+k-1}m =ka+\frac{k(k-1)}2. \]

とくに \(k\) で割った剰余に着目すると

\[ S(k,a)\equiv \frac{k(k-1)}2 \pmod{k} \]

となるので、剰余の振る舞いは \(a\) に依らず \(k\) のみによって決まる。

\(k\) \(\dfrac{k(k-1)}2 \bmod k\) 剰余 / \(k\)

30\(0\)
42\(1/2\)
50\(0\)
63\(1/2\)
\(2m\)\(m\)\(1/2\)
\(2m+1\)\(0\)\(0\)

この \(\{0,1/2\}\) という構造を \(\Q/\Z\) 的に眺めたくなるが、その前に 「\(k\) 個の和」という操作をどう拡張するか、作用素の言葉で整理しておく。

§1. 差分作用素の基礎

定義 1.1

関数 \(f:\Z\to\R\) に対して、前進差分作用素 \(\Delta\) を

\[ (\Delta f)(n)=f(n+1)-f(n) \]

で定める。

連続の微分 \(D=\dfrac{d}{dx}\) に対する離散版が \(\Delta\) である。 微分で多項式が階数を一つ下げるように、差分も適切な基底の上ではきれいに働く。

定義 1.2

下降階乗冪を

\[ x^{\underline{\beta}} := \frac{\Gamma(x+1)}{\Gamma(x+1-\beta)} \]

で定める。整数 \(\beta=m\ge 0\) なら

\[ x^{\underline m}=x(x-1)(x-2)\cdots(x-m+1) \]

となる。

命題 1.3

下降階乗冪に対しては

\[ \Delta\, x^{\underline{\beta}} = \beta\,x^{\underline{\beta-1}} \]

が成り立つ。

これは離散版の \[ \frac{d}{dx}x^\beta=\beta x^{\beta-1} \] に対応する、とても大事な公式である。

計算例 1
\[ \Delta\bigl(n^{\underline2}\bigr) = (n+1)n-n(n-1)=2n \] \[ \Delta\!\left(\frac12 n^{\underline2}\right)=n \]

したがって \(\Delta^{-1}n=\dfrac12 n^{\underline2}+C\) となる。

§2. 不定和分と連番の和

定義 2.1

\(\Delta F=f\) を満たす \(F\) を、\(f\) の不定和分と呼ぶ。

\(f(n)=n\) に対しては、さきほどの計算から

\[ \Delta^{-1}n=\frac12 n^{\underline2}+C =\frac{n(n-1)}2+C \]

である。したがって \(a\) から \(a+k-1\) までの和は

\[ \sum_{m=a}^{a+k-1}m = F(a+k)-F(a) \] \[ = \frac{(a+k)(a+k-1)}2-\frac{a(a-1)}2 = ka+\frac{k(k-1)}2. \]
計算例 2

この表示は \(k\) を整数に限らず、そのまま形式的に実数へ延長できる。 たとえば \(a=1,\;k=\dfrac32\) とすると

\[ F(x)=\frac{x(x-1)}2 \] \[ F\!\left(1+\frac32\right)-F(1) = \frac{(5/2)(3/2)}2-0 = \frac{15}{8}. \]

これは「\(\frac32\) 個の連番の和」の候補値であり、 多項式としての自然な延長になっている。

ただし、この段階ではまだ「分数個の和」が本質的に非局所になる感じは見えていない。 それが姿を現すのは、分数階差分を真正面から定義したときである。

· · ·

§3. シフト作用素と形式的べき

定義 3.1

シフト作用素 \(E\) を

\[ (Ef)(n)=f(n+1) \]

で定める。

このとき

\[ \Delta=E-I \]

である。よって形式的には

\[ \Delta^\alpha=(E-I)^\alpha \]

を考えたくなる。整数 \(\alpha=m\) なら普通の \(m\) 階差分に戻る。 しかし \(\alpha\notin\N\) では二項展開が無限和になり、非局所性が生じる。

· · ·

§4. Grünwald–Letnikov 型の分数階差分

定義 4.1

一般化二項係数を

\[ \binom{\alpha}{k} = \frac{\alpha(\alpha-1)\cdots(\alpha-k+1)}{k!} \]

とおくと、形式的には

\[ (\Delta^\alpha_{\mathrm{GL}}f)(n) = \sum_{k=0}^{\infty} (-1)^k \binom{\alpha}{k} f(n-k) \]

と書ける。

\(\alpha=m\in\N\) なら \(\binom{m}{k}=0\;(k>m)\) なので有限和で止まる。 しかし \(\alpha\notin\N\) のときは係数が永遠に消えず、無限に過去を参照する。

計算例 3: \(\alpha=\frac12\)
\[ \binom{1/2}{0}=1,\qquad \binom{1/2}{1}=\frac12,\qquad \binom{1/2}{2}=-\frac18,\qquad \binom{1/2}{3}=\frac1{16},\qquad \binom{1/2}{4}=-\frac5{128} \] \[ (-1)^k\binom{1/2}{k} = 1,\;-\frac12,\;-\frac18,\;-\frac1{16},\;-\frac5{128},\dots \]

したがって半階差分は「有限個の近傍」ではなく、無限に長い尾を持つ重み付き和になる。

注意

この GL 型の式はとても魅力的だが、増大関数に対してはそのままでは発散することがある。 たとえば \(f(n)=n\) に対しては、半階差分の級数はそのままでは収束しない。

· · ·

§5. 発散の観察：Abel 正則化

\(f(n)=n\) に対して、減衰因子 \(r^k\;(0<r<1)\) を入れて

\[ S(r) := \sum_{k=0}^{\infty} (-1)^k\binom{1/2}{k}(n-k)r^k \]

を考える。

二項級数 \[ \sum_{k=0}^{\infty}(-1)^k\binom{1/2}{k}r^k=(1-r)^{1/2} \] を使うと、

\[ \sum_{k=0}^{\infty}(-1)^k\binom{1/2}{k}k\,r^k = r\frac{d}{dr}(1-r)^{1/2} = -\frac{r}{2\sqrt{1-r}} \]

なので

\[ S(r) = n\sqrt{1-r} + \frac{r}{2\sqrt{1-r}}. \]
観察 5.1

\(r\to1^{-}\) で

\[ S(r)\sim \frac{1}{2\sqrt{1-r}} \]

となり、Abel 総和はこの場合有限値には収束しない。

つまり Abel 正則化は、ここでは 「値を与える魔法」ではなく 「どんな発散をしているかを可視化する顕微鏡」 として働く。

finite part

もし主発散項だけを引いて有限部分を取るなら

\[ \fp\lim_{r\to1^-} \left( S(r)-\frac{1}{2\sqrt{1-r}} \right) =0 \]

となる。 ただし、この \(0\) は「便利な有限部分」であって、 分数階差分の本命の値とは限らない。

§6. 本命の定義：下降階乗冪基底での分数階差分

差分作用素は下降階乗冪に対して特にきれいに振る舞う。 そこで、この基底上で直接 \(\Delta^\alpha\) を定義してしまう。

定義 6.1

このノートでは、下降階乗冪に対して

\[ \Delta^\alpha \bigl(x^{\underline{\beta}}\bigr) := \frac{\Gamma(\beta+1)}{\Gamma(\beta-\alpha+1)} x^{\underline{\beta-\alpha}} \]

と定め、線形に延長したものを分数階差分の基本定義として採用する。

この定義は整数階 \(\alpha=m\in\N\) では普通の差分と一致し、 しかも半群性

\[ \Delta^\alpha\Delta^\beta = \Delta^{\alpha+\beta} \]

を満たす。

命題 6.2

下降階乗冪基底では、分数階差分はまるで微分の解析接続版のように働く。 したがって、発散する GL 級数を無理に抱え込むより、 こちらを本体とみなすほうが理論が安定する。

§7. 実際にうまくいく計算

7.1 \(\Delta^{1/2}n\) の計算

\(n=n^{\underline1}\) なので、\(\beta=1,\;\alpha=\dfrac12\) を公式に代入すると

\[ \Delta^{1/2}n = \frac{\Gamma(2)}{\Gamma(3/2)} n^{\underline{1/2}} = \frac{2}{\sqrt{\pi}} n^{\underline{1/2}}. \]

ここで

\[ n^{\underline{1/2}} = \frac{\Gamma(n+1)}{\Gamma(n+1/2)}. \]
数値例
\[ n=1:\quad 1^{\underline{1/2}} = \frac{\Gamma(2)}{\Gamma(3/2)} = \frac{2}{\sqrt{\pi}} \] \[ \Delta^{1/2}1 = \frac{2}{\sqrt{\pi}}\cdot\frac{2}{\sqrt{\pi}} = \frac{4}{\pi} \] \[ n=2:\quad 2^{\underline{1/2}} = \frac{\Gamma(3)}{\Gamma(5/2)} = \frac{8}{3\sqrt{\pi}} \] \[ \Delta^{1/2}2 = \frac{2}{\sqrt{\pi}}\cdot \frac{8}{3\sqrt{\pi}} = \frac{16}{3\pi} \]

7.2 半階差分を二回かける

半群性により

\[ \Delta^{1/2}\Delta^{1/2}n=\Delta n=1 \]

となるはずである。実際、

\[ \Delta^{1/2}n=\frac{2}{\sqrt{\pi}}n^{\underline{1/2}} \] \[ \Delta^{1/2}\bigl(n^{\underline{1/2}}\bigr) = \frac{\Gamma(3/2)}{\Gamma(1)}n^{\underline0} = \frac{\sqrt{\pi}}2 \]

なので

\[ \Delta^{1/2}\Delta^{1/2}n = \frac{2}{\sqrt{\pi}}\cdot\frac{\sqrt{\pi}}2 = 1. \]
ここが気持ちいいところ

GL の生級数では発散が顔を出したが、下降階乗冪基底の定義では 「半階差分を二回かけると一階差分に戻る」が綺麗に実現する。 まるで作用素がちゃんと骨組みを持っている感じがする。

7.3 分数階和分 \(\Delta^{-1/2}1\)

\(\alpha=-\dfrac12,\;\beta=0\) を代入すると

\[ \Delta^{-1/2}(1) = \frac{\Gamma(1)}{\Gamma(3/2)} n^{\underline{1/2}} = \frac{2}{\sqrt{\pi}}n^{\underline{1/2}}. \]

これは半階「和分」であり、普通の \(\Delta^{-1}(1)=n+C\) の半分版だと思うと風合いがよい。

7.4 \(\Delta^{-1}n\) の再確認

\(\alpha=-1,\;\beta=1\) とすると

\[ \Delta^{-1}n = \frac{\Gamma(2)}{\Gamma(3)} n^{\underline2} = \frac12 n^{\underline2}, \]

つまり

\[ \Delta^{-1}n=\frac{n(n-1)}2+C \]

が再び得られる。整数階もちゃんと内包されている。

§8. 連番の和を有理数個へ伸ばす見方

もとの和 \[ \sum_{m=a}^{a+k-1}m \] は、\(\Delta^{-1}n\) の増分 \[ F(a+k)-F(a) \] として書けた。ここで \[ F(x)=\frac12 x^{\underline2}=\frac{x(x-1)}2. \] したがって \(k\in\Q\) や \(k\in\R\) に対しても

\[ S(k,a):=F(a+k)-F(a) \]

と定めれば

\[ S(k,a)=ka+\frac{k(k-1)}2 \]

がそのまま成立する。

計算例 4

\(a=3,\;k=\dfrac74\) とすると

\[ S\!\left(\frac74,3\right) = \frac74\cdot 3 + \frac{(7/4)(3/4)}2 = \frac{21}{4}+\frac{21}{32} = \frac{189}{32}. \]

これは「\(3\) から始まる \(\dfrac74\) 個の連番の和」の自然な延長候補である。

ただし、ここで得られたものはまだ多項式としての延長である。 一方、一般の分数階差分は本質的に非局所で、無限和を背負う。 つまり

  • 連番の和そのものは多項式的にきれいに延びる
  • 分数階差分を一般の関数に作用させると、非局所性と正則化が現れる

という二つの顔がある。

§9. RL 型と Caputo 型のひとこと

連続の分数微分でもそうだが、離散でも「どの定義を採るか」で定数の扱いが少し変わる。 このノートの定義は Riemann–Liouville 型に近く、たとえば一般には \[ \Delta^{1/2}(1)\neq 0 \] である。

もし「定数の分数階微分も 0 であってほしい」という感覚を優先するなら、 Caputo 型の修正版を使う流儀もある。 ただし今回のノートでは、 下降階乗冪基底での代数的な美しさを優先して RL 型の考え方を採っている。

§10. まとめ

まとめ

1. 整数個の連番の和は \[ \sum_{m=a}^{a+k-1}m = ka+\frac{k(k-1)}2 \] であり、これは \(k\) を実数へそのまま延長できる。

2. 分数階差分を GL 型の生級数で書くと、 \(f(n)=n\) のような増大関数では発散する。 Abel 正則化は、この発散が \[ \sim \frac{1}{2\sqrt{1-r}} \] であることを見せてくれる。

3. 実際に安定した値を与えるには、 \[ \Delta^\alpha\bigl(x^{\underline\beta}\bigr) = \frac{\Gamma(\beta+1)}{\Gamma(\beta-\alpha+1)} x^{\underline{\beta-\alpha}} \] を基本定義とするのがきれいである。

4. その結果、 \[ \Delta^{1/2}n = \frac{2}{\sqrt{\pi}}\,n^{\underline{1/2}} \] となり、さらに \[ \Delta^{1/2}\Delta^{1/2}n=1 \] も美しく成立する。

言い換えると、今回必要だった正則化法は 「Abel そのもの」ではなく、 下降階乗冪基底へ移して Gamma 関数で解析接続する方法 だった、ということになる。
